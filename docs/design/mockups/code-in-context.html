CODE = B (LEFT RULE) Manifold Learning

Isomap

Recover a low-dimensional embedding by preserving geodesic distances along the data manifold.

Geodesic distances

Straight-line distance cuts across the fold, so Isomap instead measures distance along the manifold: build a neighborhood graph, then take shortest paths. Set the neighbor count with k = 10.

Note
Too small a k fragments the graph; too large a k short-circuits across the fold.

The geodesic between two points is approximated by the graph shortest path:

Dij = minpaths ∑ ||xa − xb||(3)

Double-centering this distance matrix and taking the top eigenvectors yields the embedding.

Full pseudocode
1function isomap(X, k, d):
2    G = knn_graph(X, k)            # k nearest neighbors
3    D = shortest_paths(G)         # geodesic distances
4    B = -0.5 * J @ (D ** 2) @ J   # double centering
5    L, V = eig(B)                  # top eigenpairs
6    return V[:, :d] * sqrt(L[:d])  # embedding
